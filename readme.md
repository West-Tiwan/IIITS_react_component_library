# IIITS Form Library

This is a simple form library developed at IIITS.

## Installation

You can install the `iiits-form-library` via npm:

```bash
npm install iiits-form-library
````
## Usage

```javascript

import { Form } from 'iiits-form-library';

// Use the Form component
````
